# X Announcement: CURATE Launch

## Post 1: The AI Data Problem
Autonomous AI agents are trading billions of dollars, but they rely on unbonded data APIs. If an oracle lies, who pays?
Introducing **CURATE** — the first economic stacking bond protocol for data oracles. Built on Hacash. 🧵👇

## Post 2: The Physical Firewall
To activate a CURATE oracle passport, providers must lock 75 HAC + 1 HACD in L1 stack escrow. 
If they feed false data, their passport is burned and their 75 HAC collateral is locked in the Hacash Penalty Pool for 120 days.

## Post 3: Token Architecture
No pre-mine. No team reserves. 
- **SFT Passport:** The active validator key.
- **CRT Token (FT):** Curation credits to pay for data queries and stake on oracle indexes.
