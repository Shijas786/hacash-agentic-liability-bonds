# Project Profile: CURATE

## Metadata
- **Project Name:** CURATE
- **Ticker:** CRT
- **Category:** AI Curation & Oracle Registry
- **Asset Type:** HYBRID (SFT Passport + FT Curation Credits)
- **HACD Lots:** 400
- **Collateral Cost:** 75 HAC per lot
- **Units per Lot:** 16,777,216 CRT
- **Total Supply:** 6,710,886,400 CRT
- **Total Formation Cost:** 30,000 HAC locked
- **Removal Effect:** Burn (permanent destruction of identity SFT)

## Description
CURATE is a PoW-backed economic bonding and slashing middleware protocol for AI data providers, API indexes, and oracle nodes on the Hacash Stack Protocol.

## Utility Design
- **CURATE Passport (SFT):** Minted upon stacking 75 HAC + 1 HACD. Acts as the active identity token for an API node. Burned permanently on-chain if node submits false data feeds.
- **CRT Curation Token (FT):** Fungible utility credits used by consuming AI models to pay for data queries, stake on oracle accuracy indexes, and vote on dispute resolutions.
