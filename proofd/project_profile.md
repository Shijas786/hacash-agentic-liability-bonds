## The Problem PROOFD Solves

As autonomous AI agents execute complex financial transactions, manage vaults, and make governance decisions, the industry face a critical vulnerability: **lack of economic accountability**.
- **Malicious Agent Exploits:** Agents can exploit DeFi protocols or run frontrunning attacks. If they succeed, they profit; if they fail, they lose nothing because their identity is throwaway.
- **Zero-Cost Sybil Attacks:** Attackers can spin up 10,000 clone agents to flood orderbooks or spam oracle feeds with no skin in the game.
- **Protocol Default Risks:** Autonomous agents running yield strategies or margin trades have no collateral backing their actions, making them unsafe counterparties.

PROOFD resolves this by requiring active agents to lock a **PROOFD Security Bond** (1 HACD + 50 HAC). If an agent misbehaves, it is slashed: the Passport is burned, the HACD is blacklisted, and the 50 HAC stack is locked for a long penalty cooling period, ensuring agent operations carry real liability.
