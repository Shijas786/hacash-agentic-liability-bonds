## What Is PROOFD?

AI agents manage millions in TVL, execute swaps, and route transactions. But they carry zero legal or financial liability. If an agent executes an exploit or defaults on a trade, it loses nothing. 

PROOFD introduces **Proof-of-Agent-Work Collateral Bonds**.

By stacking 50 HAC onto a scarce, PoW-mined HACD, an operator locks a $2,500+ economic security bond. If the agent misbehaves (e.g. oracle manipulation, flash loan exploits, trade default), the decentralized **Slashing Oracle** burns the agent's Passport SFT and locks the 50 HAC collateral for a 90-day cooling freeze.

## The PROOFD Passport SFT

Each PROOFD Passport is:

- **Collateral-Backed** — represents active on-chain security backing (50 HAC + HACD).
- **Non-transferable** while stacked, binding the liability to the operator's wallet.
- **Slashable** — if the agent misbehaves, the SFT is burned and collateral is frozen.
- **De-bondable** — voluntary removal returns the HACD and 50 HAC instantly (if no pending slash disputes exist).
