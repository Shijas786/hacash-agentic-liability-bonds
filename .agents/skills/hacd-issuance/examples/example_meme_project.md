# Example: Meme Stack Project

## One-liner
A meme asset formed through HACD Stack, where each HACD carries a fixed amount of the meme supply.

## Why HACD

Most meme tokens are deployed instantly. A HACD meme can be formed through visible PoW-backed Stack lots, making the launch process itself part of the culture.

## Example structure

- Asset type: FT
- Total HACD lots: 256
- Units per HACD: 65,536,000
- Total supply: 16,777,216,000
- Stack cost: 88 HAC per HACD
- Removal effect: token amount associated with that HACD is removed / burned according to final protocol rules

## Launchpad angle

Formation, not just issuance.

Each stacked HACD carries a visible piece of the meme supply.
