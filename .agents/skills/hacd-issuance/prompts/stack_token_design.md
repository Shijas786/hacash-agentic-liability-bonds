# Prompt: Stack Token Design

Design the HACD Stack issuance structure for the project.

Return:

1. Asset type
2. Total supply
3. HACD lot structure
4. Units per HACD
5. Stack cost per HACD in HAC
6. Estimated total formation cost
7. Launch phases
8. Removal / burn logic
9. Participant flow
10. Open questions

Use simple math and show formulas.

Do not claim that stack cost creates a guaranteed floor price. Use `formation cost reference` instead.
