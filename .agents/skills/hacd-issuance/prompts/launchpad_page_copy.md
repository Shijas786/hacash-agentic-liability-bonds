# Prompt: Launchpad Page Copy

Write Launchpad-ready copy for a HACD Stack Asset.

Structure:

1. Hero headline
2. Subheadline
3. What is this asset?
4. Why HACD?
5. How Stack works
6. Launch details
7. Utility
8. FAQ
9. Risk disclosure

Style:

- clear
- beginner-friendly
- official
- not overhyped
- no profit promises
- no misleading price floor language

Include both technical clarity and narrative strength.
