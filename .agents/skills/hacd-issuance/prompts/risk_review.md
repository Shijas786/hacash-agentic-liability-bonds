# Prompt: Risk Review

Review the issuance package for clarity and safety.

Flag:

- profit promises
- misleading price floor claims
- unclear HACD / HAC requirements
- missing supply math
- missing launch rules
- unsupported utility claims
- regulatory-sensitive claims
- private key or seed phrase requests
- centralized custody risks

Return:

1. Critical issues
2. Recommended changes
3. Safer wording
4. Final reviewer checklist
